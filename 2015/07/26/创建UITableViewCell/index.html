创建UITableViewCell子类

概述：
UITableView对象显示的是一组UITableViewCell对象，自带的属性有：textLabel, detailTextLabel和imageView。
如果需要UITableViewCell对象显示更多细节，或者拥有不同的布局时，需要创建自定义的UITableViewCell子类。

  • 效果图：
  1. UITableViewCell是UIView的子类，定制界面－>覆盖drawRect
    contentView：UITableViewCell的子视图，起容器的作用，将定制的修改的UITextField、UILabel和UIButton等视图对象加入contentView，可以改变UITableViewCell外观。
  2. 处理图片
    UITableViewCell采用缩略图显示图片，thumbnail，

  3. 向UITableViewCell上添加UIControl对象
    视图对象，不应该直接存取模型对象，所以，通过视图控制器对象，添加一个Block对象。

    引入学习Block对象
    void (^)(NSString *someArg, int anotherArg)
    Block在栈中创建，其他对象在堆中，copy某个Block对象时，应用会在堆中创建该对象的备份。

  4. 捕获变量
    Block对象可以使用其封闭作用域内的所有变量。

深入学习：UICollectionView

  • UICollectionView是UIScrollView的子类
  • 与UITableViewCell类似，UICollectionView对象显示一组UICollectionViewCell
  • UICollectionView具有数据源
